üß© Problem Statement

Design and implement a network for a company located in a three stored building. The first floor has three departments; reception, stores, and logistics, the second floor also has three departments; finance, HR, and sales, and finally on the third floor there are two departments; Admin and IT. The following are part of consideration during implementation:

  • Each floor should have a router all placed in the server room of IT department.
  • All routers should be connected to each other using serial DCE cable.
  • Network between routers should be 10.10.10.0/30, 10.10.10.4/30, 10.10.10.8/30
  • Each floor is expected to have one switch placed in respective floors.
  • Each floor is expected to have WI-FI network for laptops and phones
  • Each department is expected to have a printer.
  • Each department is expected to be in a different VLAN with the following details:
      • 1st Floor; Reception: VLAN 80 & IP 192.168..8.0/24, Store: VLAN 70 & IP 192.168.7.0/24, Logistics: VLAN 60 & IP 192.168.6.0/24
      2nd Floor; Finance: VLAN 50 & IP 192.168.5.0/24, HR: VLAN 40 & IP 192.1168.4.0/24, Sales: VLAN 30 & 192.168.3.0/24
      3rd Floor; Admin VLAN 20 & IP 192.168.2.0/24, IT: VLAN 10 & IP 192.168.1.0/24
  • Use OSPF as routing protocol to advertise routers
  • All devices in the network are expected to obtain IP address dynamically with their repsective router configured as their DHCP router.
  • Configure SSH in all routers for remote login
  • In IT department, add a PC called Test-PC to port fa0/1 and use it to test remote login
  • Configure port security to IT-department switch to allow only Test-PC to access port fa0/1 (use sticky method to obtain mac-address with violation mode of shutdown)

üîê Network Security Suggestions

  • Implement ACLs to restrict inter-VLAN traffic.
  • Use strong passwords and enable password encryption.
  • Disable unused ports and enable port security.
  • Configure SSH instead of Telnet for remote access.
  • Regularly update firmware and monitor logs.
